# SQL Queries

This folder contains all SQL scripts used for analyzing the 2026 Tamil Nadu Assembly Election dataset.

## Analysis Performed

### Party Vote Share Analysis
- Analyzed total votes received by each political party.
- Identified vote share distribution across parties.

### Political Swing Analysis (2021 vs 2026)
- Compared seat performance between 2021 and 2026 elections.
- Identified major gainers and losers.

### Region-wise Party Dominance
- Evaluated party performance across different regions.
- Identified regional strongholds and leading parties.

### Region-wise Voter Turnout
- Calculated average voter turnout by region.
- Compared voter participation patterns.

### Top 10 Closest Election Contests
- Identified constituencies with the smallest victory margins.
- Highlighted the most competitive races.

### Top 10 Highest Vote-Getting Candidates
- Ranked candidates based on total votes received.
- Identified the most popular candidates.

### Top 10 Largest Victory Margins
- Calculated the highest winning margins.
- Identified constituencies with dominant victories.

## SQL Concepts Used

- Aggregations
- GROUP BY
- ORDER BY
- JOINs
- Common Table Expressions (CTEs)
- Window Functions
- ROW_NUMBER()

These queries served as the foundation for generating insights that were later visualized in Power BI and presented through data storytelling.
